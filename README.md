## Kurulum

1. Repo'yu klonlayın:
   ```bash
   git clone https://github.com/elifgider/my_portfolio.git
   ```
2. Proje dizinine girin:
   ```bash
   cd my_portfolio
   ```
3. Gerekli bağımlılıkları yükleyin (örn. Node.js ile):
   ```bash
   npm install
   ```
4. Uygulamayı başlatın:
   ```bash
   npm start
   ```
5. Tarayıcıda `http://localhost:3000` adresinden görüntüleyebilirsiniz.

## Kullanılan Teknolojiler

- Next.js (App Router): Dosya tabanlı yönlendirme, sunucu bileşenleri
- React 18: Bileşen tabanlı UI
- TypeScript: Tür güvenliği ve daha iyi DX
- Tailwind CSS: Hızlı, utility-first stil yazımı
- PostCSS + Autoprefixer: CSS işleme ve tarayıcı uyumluluğu
- MDX: Blog ve proje içeriklerini .mdx ile yazma
- ESLint: Kod kalitesi ve stil denetimi
- Node.js + npm: Geliştirme ve paket yönetimi
- Netlify (opsiyonel): Dağıtım/hosting
