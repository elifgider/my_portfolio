## NPX İle Kişisel Kartvizit Paketi Nasıl Yapılır
Terminal'de `npx elif` yazıp Enter'a bastığınızda benim kartvizitim çıkıyor! Eğer bu havalıysa ve siz de kendi npx paketinizi yapmak istiyorsanız, bu yazı tam size göre! 🚀

---

## 🛠️ Adım 1: Proje Oluşturma

```bash
mkdir my-card
cd my-card
npm init -y
```

---

## 📦 Adım 2: Package.json Düzenleme

`package.json` dosyasını açıp şu değişiklikleri yapın:

```json
{
  "name": "my-card",
  "version": "1.0.0",
  "description": "Kişisel npx kartviziti",
  "main": "index.js",
  "bin": {
    "my-card": "./index.js"
  },
  "keywords": ["cli", "npx", "kartvizit"],
  "author": "Your Name",
  "license": "MIT"
}
```

**Önemli:** `bin` field'i hangi dosyanın çalıştırılacağını belirtir.

---

## 🚀 Adım 3: Ana Dosya Oluşturma

`index.js` dosyasını oluşturun:

```javascript
#!/usr/bin/env node

console.log(`
╔══════════════════════════════════════╗
║                                      ║
║  Your Name                           ║
║  Your Title                          ║
║                                      ║
║  📧 your@email.com                   ║
║  🐙 github.com/yourusername          ║
║  💼 linkedin.com/in/yourusername     ║
║  🌐 yourwebsite.com                  ║
║                                      ║
║  Kartvizit: npx my-card              ║
║                                      ║
╚══════════════════════════════════════╝
`);

console.log('Merhaba! Ben [Your Name]. [Your Title] olarak çalışıyorum.');
```

**Dosyanın başındaki `#!/usr/bin/env node` çok önemli!**

---

## 🔧 Adım 4: Çalıştırılabilir Yapma

```bash
chmod +x index.js
```

---

## 🧪 Adım 5: Test

```bash
node index.js
```

---

## 📤 Adım 6: NPM'e Yayınlama

```bash
npm login
npm publish
```

---

## 🎉 Tamamlandı!

Artık dünyanın her yerinden herkes şu komutu çalıştırabilir:

```bash
npx my-card
```

---

## 💡 İpuçları

- Paket adı benzersiz olmalı
- `package.json`'da `bin` field'ini unutmayın
- `#!/usr/bin/env node` ile dosyayı başlatın
- `chmod +x` ile çalıştırılabilir yapın
